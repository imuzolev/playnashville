## Команды на сервере

### Перезапуск приложения
> ⚠️ **Обязательно** после любых изменений!

```bash
sudo systemctl restart chords_app
```

### Права на статику
После добавления новых файлов в static:
```bash
chmod 755 /root/chords_app/static
chmod 644 /root/chords_app/static/*
```

### Статус приложения
```bash
sudo systemctl status chords_app
```

### Логи ошибок
```bash
sudo journalctl -u chords_app -f
```

